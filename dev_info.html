3 Hal Yang Paling sulit saya implementasikan dalam membuat web ini namun masih bisa saya selesaikan:

1. Membuat Tampilan Web yang memiliki tab-tab seperti model flexbox namun bertabrakan dengan implementasi Materialize yang kurang support membuat tampilan seperti ini

2. Membuat permodelan Responsive Web pages yang cukup menyulitkan dikarenakan Navbar dari Materialize menghilang jika ukuran layar kurang dari 641px

3. Pengeditan margin yang cukup menyulitkan dalam membuat web yang presisi seperti ini, dikarenakan harus memikirkan lebih jauh bagaimana pengimplementasiannya kedalam responsive web pages

1 Element yang ingin saya peduhi pada tampilan web ini namun mustahil dalam saya pecahkan :

Saya ingin mendesain tampilan web yang terlihat aktif kepada pengguna dengan cara mengimplementasikan Javascript pada web ini namun saya belum memiliki kemampuan sejauh itu.

Bagian yang menurut saya Sulit(hard) serta Mustahil(impossible) saya wujudkan dalam mendesain web ini:

Hard Case :

Mengimplementasikan teknik Responsive Web Pages pada seluruh halaman web ini.

Impossible Case :

Memberikan tampilan loading pages modern yang seakan-akan jika kita mengklik halaman selanjutnya, web akan merespon dengan menampilkan loading screen yang telah dirancang sedemikian rupa. pada bagian ini teknik javascript lanjutan diperlukan dalam merancangnya
